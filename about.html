About this website

How to make your own personal website (like this one):

For accessibility, I have written with the assumption that the reader has little or no knowledge of web development.

Before starting, you should have the following:

  • A Github account. Github Pages will allow you to publish and maintain a static website hosted in a Github repository for free.
  • A Formspree account. Formspree makes it simple to set up and receive responses from forms on your website without requiring knowledge of backend (server-side) web development. If you donâ€™t plan on having forms on your website, this is unnecessary.
  • Some basic understanding of HTML/CSS, to customize a template (if you choose to use one) or create the necessary files for the website from scratch.
